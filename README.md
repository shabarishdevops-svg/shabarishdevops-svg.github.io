## 🚀 Local Development

### Prerequisites

- Node.js (v16 or higher)
- Yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/shabarishdevops-svg/shabarishdevops-svg.github.io.git

# Navigate to the directory
cd shabarishdevops-svg.github.io

# Install dependencies
yarn install

# Start development server
yarn start
```

The site will open at `http://localhost:3000`

### Build for Production

```bash
# Create optimized production build
yarn build
```

The build files will be in the `build/` directory.

## 📦 Deployment

This site is deployed using GitHub Pages. To deploy updates:

```bash
# Install gh-pages (first time only)
yarn add --dev gh-pages

# Deploy to GitHub Pages
yarn deploy
```

## 🎨 Customization

### Color Scheme

The portfolio uses a professional dark theme with the following colors:

- **Background:** `#0a0e17` (Dark Navy)
- **Secondary Background:** `#0f1419` (Darker Navy)
- **Accent:** `#60a5fa` (Blue)
- **Text Primary:** `#ffffff` (White)
- **Text Secondary:** `#a1a1aa` (Gray)

To customize, edit the CSS files in `/src/components/`

### Content Updates

- **Personal Info:** Edit `/src/components/Hero.jsx`
- **About Section:** Edit `/src/components/About.jsx`
- **Skills:** Edit `/src/components/Skills.jsx`
- **Contact Info:** Edit `/src/components/Contact.jsx`
